  1. git clone url -> clone repository dari git
  2. touch hello.html -> create file
  3. git add hello.html -> langkah awal sebelum commit ke git online repository
  4. ls -> cek jika file sudah di snapshot untuk siap di commit

  6. git status -> cek status sebelum di push ke online repository
  7. git push -> upload/push file commit ke online repository
  8. git commit -am "message" -> add dan commit secara instant, cara cepat, tinggal push
